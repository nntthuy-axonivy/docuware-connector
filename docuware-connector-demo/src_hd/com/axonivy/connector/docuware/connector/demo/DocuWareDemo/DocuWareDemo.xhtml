Basic DocuWare calls

This demo allows you to connect and play around with one or many DocuWare configurations.

Open document #{data.ctrl.documentId} in new tab
Viewer will be enabled for configurations containing an integration passphrase.
Set a document Id to enable document related functions.
Set fields. Notes: use technical field names which can be found by inspecting available fields of the cabinet.
Set a file cabinet Id to fetch documents and enable file upload.
Here you can create a login token for the configured DocuWare instance OR you can enter a token if you got it by other means. This login token can then be set for the selected configuration. Note, that only configurations with grantType dwtoken can be selected. After the token is set, the configuration should work in a normal way.
